# Task 2 - Моделирование динамики спортивных матчей

[← К главному README](../README.md)

Подпроект находится в папке [`modeling_match_dynamics`](./modeling_match_dynamics/README.md).

Кратко: задача моделирует динамику футбольных и NBA-матчей по последовательностям событий. Внутри реализованы Football Events pipeline для прогноза голов во втором тайме и NBA Player Tracking pipeline для прогноза изменения разницы счета за последние 5 минут.

Основная документация: [Modeling Match Dynamics](./modeling_match_dynamics/README.md).
