# MOS — инструкция для оценщика

Оцениваются автоматически сгенерированные описания футбольных матчей по шкале
**Mean Opinion Score (1..5)**. В `mos_questionnaire.csv` — 20 образцов, источник
модели скрыт (слепая оценка), истинная модель — в `mos_key.json`.

По каждому образцу три оценки 1..5:

- **fluency** — грамматичность и беглость английского текста;
- **coherence** — логическая связность предложений;
- **realism** — насколько текст похож на настоящую ленту событий матча.

Каждый оценщик сохраняет файл `mos_ratings_<имя>.csv`. Агрегатор усредняет по
всем таким файлам и пишет `results/tables/mos_results.*`. В этом прогоне оценку
дали 5 оценщиков с разными профилями (строгость к грамматике / связности /
реалистичности).
